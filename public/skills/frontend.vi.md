# Frontend Development

Đây là thế mạnh lớn nhất của tôi: xây dựng giao diện web hiện đại, nhanh, dễ bảo trì và chú trọng trải nghiệm người dùng.

## React.js — *Expert*

React là công cụ tôi dùng hằng ngày. Tôi thành thạo:

- Kiến trúc component theo hướng tái sử dụng, tách biệt rõ ràng giữa UI và logic.
- Hooks (`useState`, `useEffect`, `useMemo`, custom hooks) và quản lý state hợp lý.
- Tối ưu hiệu năng: memo hoá, lazy-loading, code-splitting.
- Tích hợp AI chạy ngay trong trình duyệt (chính website này là một ví dụ).

## TypeScript — *Advanced*

Tôi viết toàn bộ dự án bằng TypeScript để có type an toàn:

- Định nghĩa `interface` / `type` rõ ràng, hạn chế lỗi runtime.
- Discriminated union, generics cho các API tái sử dụng.

## Next.js — *Advanced*

- Server-side rendering và static generation cho SEO tốt hơn.
- File-based routing, API routes.

## HTML / CSS / JavaScript — *Advanced*

- HTML ngữ nghĩa, accessibility (a11y).
- CSS hiện đại với Tailwind, Flexbox/Grid, responsive design.

## Angular — *Intermediate*

- Component, service, dependency injection.
- Angular Material cho hệ thống giao diện nhất quán.

---

> Triết lý của tôi: giao diện đẹp là chưa đủ — nó phải nhanh, dễ tiếp cận và dễ bảo trì cho cả team.
