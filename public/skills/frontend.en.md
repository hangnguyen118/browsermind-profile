# Frontend Development

This is my strongest area: building modern, fast, maintainable web interfaces with a focus on user experience.

## React.js — *Expert*

React is my daily driver. I'm confident with:

- Reusable, composable component architecture with a clear UI/logic separation.
- Hooks (`useState`, `useEffect`, `useMemo`, custom hooks) and sensible state management.
- Performance tuning: memoization, lazy-loading, code-splitting.

## TypeScript — *Advanced*

I build entire projects in TypeScript for type safety:

- Clear `interface` / `type` definitions that cut down on runtime errors.
- Discriminated unions and generics for reusable APIs.

## Next.js — *Advanced*

- Server-side rendering and static generation for better SEO.
- File-based routing and API routes.

## HTML / CSS / JavaScript — *Advanced*

- Semantic HTML and accessibility.
- Modern CSS with Tailwind, Flexbox/Grid, responsive design.

## Angular — *Intermediate*

- Components, services, dependency injection.
- Angular Material for a consistent design system.

---

> My philosophy: a beautiful UI isn't enough — it has to be fast, accessible, and maintainable for the whole team.
